---
title: 'Installation'
description: How to install @constela/server
---

## Package Manager

```bash
npm install @constela/server
```

```bash
pnpm add @constela/server
```

```bash
yarn add @constela/server
```

## Peer Dependencies

<Callout type="note">
**Requirements**: `@constela/compiler` (peer dependency)
</Callout>

```bash
npm install @constela/compiler @constela/server
```
