---
title: 'Examples'
description: Usage examples for @constela/cli
---

## Build a Program

```bash
constela build app.json --output dist/app.js --minify
```

## Start Development Server

```bash
constela dev app.json --port 3000 --open
```

## Validate All Files in a Directory

```bash
constela validate --all src/
```

## Inspect Program Structure

```bash
constela inspect app.json --state --json
```

## Initialize a New Project

```bash
constela init my-app
cd my-app
npm install
npm run dev
```
