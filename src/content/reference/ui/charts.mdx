# Charts

A comprehensive charting component supporting 12 chart types with animations, tooltips, and interactivity.

## Basic Usage

```json
{
  "kind": "component",
  "name": "Chart",
  "props": {
    "type": { "expr": "lit", "value": "line" },
    "data": { "expr": "state", "name": "chartData" }
  }
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `Expression` | Required | Chart type |
| `data` | `Expression` | Required | Chart data |
| `options` | `Expression` | `{}` | Chart options |
| `width` | `Expression` | `"100%"` | Chart width |
| `height` | `Expression` | `400` | Chart height |
| `onClick` | `EventHandler` | - | Click handler |
| `onHover` | `EventHandler` | - | Hover handler |

## Chart Types

| Type | Description |
|------|-------------|
| `line` | Line chart |
| `bar` | Vertical bar chart |
| `horizontalBar` | Horizontal bar chart |
| `pie` | Pie chart |
| `doughnut` | Doughnut chart |
| `area` | Area chart |
| `scatter` | Scatter plot |
| `radar` | Radar/spider chart |
| `polar` | Polar area chart |
| `bubble` | Bubble chart |
| `histogram` | Histogram |
| `candlestick` | Candlestick chart |
| `treemap` | Treemap visualization |

## Data Structure

```typescript
interface ChartData {
  labels: string[];
  datasets: Dataset[];
}

interface Dataset {
  label: string;
  data: number[] | Point[];
  backgroundColor?: string | string[];
  borderColor?: string;
  borderWidth?: number;
  fill?: boolean;
  tension?: number; // For curved lines
}

interface Point {
  x: number;
  y: number;
  r?: number; // For bubble charts
}
```

## Examples

### Line Chart

```json
{
  "version": "1.0",
  "state": {
    "chartData": {
      "type": "object",
      "initial": {
        "labels": ["Jan", "Feb", "Mar", "Apr", "May", "Jun"],
        "datasets": [{
          "label": "Sales",
          "data": [65, 59, 80, 81, 56, 55],
          "borderColor": "hsl(220, 90%, 56%)",
          "tension": 0.4
        }]
      }
    }
  },
  "view": {
    "kind": "component",
    "name": "Chart",
    "props": {
      "type": { "expr": "lit", "value": "line" },
      "data": { "expr": "state", "name": "chartData" },
      "options": { "expr": "lit", "value": {
        "responsive": true,
        "animation": { "duration": 750, "easing": "easeOutQuart" },
        "plugins": {
          "legend": { "position": "top" },
          "tooltip": { "enabled": true }
        },
        "scales": {
          "y": { "beginAtZero": true }
        }
      }}
    }
  }
}
```

### Real-time Updates

```json
{
  "version": "1.0",
  "state": {
    "chartData": { "type": "object", "initial": { "labels": [], "datasets": [{ "label": "CPU", "data": [] }] } }
  },
  "lifecycle": { "onMount": "startMonitoring" },
  "actions": [
    {
      "name": "startMonitoring",
      "steps": [{
        "do": "interval",
        "ms": { "expr": "lit", "value": 1000 },
        "action": "updateChart"
      }]
    },
    {
      "name": "updateChart",
      "steps": [{
        "do": "fetch",
        "url": { "expr": "lit", "value": "/api/metrics" },
        "onSuccess": [{
          "do": "setPath",
          "target": "chartData",
          "path": { "expr": "lit", "value": ["datasets", 0, "data"] },
          "value": { "expr": "var", "name": "response", "path": "cpuHistory" }
        }]
      }]
    }
  ],
  "view": {
    "kind": "component",
    "name": "Chart",
    "props": {
      "type": { "expr": "lit", "value": "line" },
      "data": { "expr": "state", "name": "chartData" },
      "options": { "expr": "lit", "value": {
        "animation": false,
        "scales": { "x": { "display": false } }
      }}
    }
  }
}
```

## Chart Options

### Animation

```json
{
  "animation": {
    "duration": 1000,
    "easing": "easeOutQuart",
    "delay": 0
  }
}
```

**Easing functions:** `linear`, `easeInQuad`, `easeOutQuad`, `easeInOutQuad`, `easeInCubic`, `easeOutCubic`, `easeInQuart`, `easeOutQuart`, `easeInOutQuart`

### Scales

```json
{
  "scales": {
    "x": {
      "display": true,
      "title": { "display": true, "text": "Month" }
    },
    "y": {
      "beginAtZero": true,
      "max": 100,
      "title": { "display": true, "text": "Value" }
    }
  }
}
```

### Plugins

```json
{
  "plugins": {
    "legend": {
      "display": true,
      "position": "top"
    },
    "tooltip": {
      "enabled": true,
      "mode": "index"
    },
    "title": {
      "display": true,
      "text": "Monthly Sales"
    }
  }
}
```

## Styling

CSS classes:

- `.chart-container` - Main container
- `.chart-canvas` - Canvas element
- `.chart-legend` - Legend container
- `.chart-tooltip` - Tooltip container
- `.chart-loading` - Loading state

## Accessibility

- ARIA labels for chart elements
- Keyboard navigation for data points
- High contrast mode support
- Screen reader descriptions for data
